# 🏦 Bank Management System (Flask Project)

A full-stack Bank Management System built using **Flask (Python)** for backend and **HTML, CSS, JavaScript** for frontend.

This project allows users to create accounts, check balances, and perform transactions like deposit, withdraw, and transfer.

---

## 🚀 Features

- 🔐 User Registration & Login (JWT Authentication)
- 🏦 Create Bank Account
- 💰 Check Account Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- 🔄 Transfer Money Between Accounts
- 📜 Transaction History
- 🌐 REST API with Flask
- 🎨 Responsive Frontend UI
